2025 = 行动的一年

前年就开始了解AI，意识到这是要抓住的时代机遇，但停留在看视频——后来才发现，看视频是一种懒惰，因为不敢采取行动才会去学很多东西。

今年真正骄傲的事情：内容突破（中英文）、转产品岗、发布第一个产品、动手Vibe Coding。行动过程中学到的东西，远远大于看书看视频。

AI时代的营销变难

产品泛滥 → 注意力稀缺

以前：代码是稀缺资源
现在：注意力才是稀缺资源

营销和分发成为壁垒。创始人必须是CMO，个人品牌与产品品牌不分家。

活人感 = 未来关键词

社交媒体上充斥着大量AI生成的内容，非常不真诚。

脱颖而出的内容：有活人感的内容

  • 播客（有瑕疵的声音）
  • 出镜视频（快速建立连接）
  • 全员Marketing

冷启动：5%的改变

01. 从问题出发

回想周围人最常问你的三个问题——这就是你最有趣的地方，最适合做成内容。

02. 降低门槛

设备、场地、剪辑都不重要。拿着手机就开始讲，大白话讲出来。

03. 重复再重复

当你觉得讲吐了，大家才刚刚开始听。默认情况就是nobody cares。

文科生的超能力

Technical Curiosity（科技好奇心）是可以习得的技能，不是天生标签。

AI时代大家都在同一起跑线，关键是：

  • Idea（独特洞察）
  • Distribution（分发能力）
  • Agency（主动性）

Code is Cheap, Show Me the Talk

以前：Talk is cheap, show me the code

现在：Code is cheap, show me the talk

做什么（What）比怎么做（How）更重要。文科生更能回答做什么，更能讲好产品故事。
